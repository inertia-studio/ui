# Changelog

All notable changes to `@inertia-studio/ui` will be documented in this file.

## [0.1.0] — 2026-04-05

Initial release.

### Added

- **30 field components** — TextField, EmailField, PasswordField, UrlField, TelField, NumberField, StepperField, TextareaField, SelectField, ToggleField, CheckboxField, CheckboxListField, RadioField, ToggleButtonsField, DateField, TimeField, DateRangeField, ColorPickerField, FileUploadField, ImageUploadField, TagsField, KeyValueField, RepeaterField, HiddenField, PlaceholderField, SlugField, MoneyField, PercentField, BelongsToField, RichEditorField
- **8 table cell renderers** — TextCell, BadgeCell, BooleanCell, ImageCell, IconCell, ColorCell, DateCell, MoneyCell
- **4 filter components** — SelectFilter, TernaryFilter, DateFilter, BooleanFilter
- **6 detail entry renderers** — TextEntry, BadgeEntry, BooleanEntry, ImageEntry, DateEntry, MoneyEntry
- **DataTable** — Sortable, searchable, paginated data table with column toggle and bulk action bar
- **InlineTable** — Self-contained table with client-side search, sort, and pagination
- **InlineForm** — Standalone form for custom pages
- **FormRenderer** — Schema-driven form with sections, tabs, field wrappers, and validation display
- **SimpleFormModal** — Modal-based form for simple modules
- **DetailView** — Read-only record display with entry renderers
- **Layout components** — PanelLayout, Sidebar, Topbar, Breadcrumbs, UserMenu, GlobalSearch
- **Action components** — ActionButton, ActionDropdown, ConfirmationModal
- **Tab components** — TabGroup, TabPanel
- **Toast notifications** — Flash-based toasts with progress bar and icons
- **LoadingBar** — Thin accent progress bar on Inertia navigation
- **DatePicker** — Custom single date picker (zero dependencies)
- **DateRangePicker** — Date range picker with presets
- **Select** — Custom select with search, multi-select, server-side search
- **Icon** — Icon renderer with provider system
- **8 default pages** — ListPage, CreatePage, EditPage, ViewPage, LoginPage, ProfilePage, DashboardPage (with SchemaRenderer for 24 primitives)
- **4 hooks** — usePanel, useFormState, useTheme, useStudioHooks
- **Design system** — Semantic color tokens (`s-bg`, `s-surface`, `s-text`, `s-accent`, etc.), dark mode, 10 theme presets
- **Tailwind preset** — Custom theme with Studio variables
